# Pocket Desktop Prototype (HeyPocket)

This is a technical proof-of-concept built specifically for the **Software Engineer (Desktop)** role at **Pocket (YC W26)**.

## Project Goal

The objective of this prototype is to demonstrate proficiency in **Tauri 2.0**, **Rust**, and **React** by building a cross-platform desktop application that mirrors the core requirements of the HeyPocket ecosystem.

## Key Features Implemented

- **Rust Backend Commands:** Secure Inter-Process Communication (IPC) between Rust and the frontend.
- **Hardware Simulation:** A Rust-based command that simulates the detection and pairing of the HeyPocket device over USB/BLE.
- **Real-time Sync UI:** A modern, responsive dashboard built with React and Tailwind CSS that simulates meeting transcript synchronization.
- **Tauri 2.x Architecture:** Utilization of the latest Tauri 2.0 standards for cross-platform performance.

## Tech Stack

- **Backend:** Rust (Tauri 2.x)
- **Frontend:** React + TypeScript
- **Styling:** Tailwind CSS / Modern UI

## How to run (Development)

1. Ensure you have [Rust](https://rustup.rs/) and [Node.js](https://nodejs.org/) installed.
2. Clone this repository.
3. Run `npm install`.
4. Run `npm run tauri dev`.
